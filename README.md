# 出院轉銜管理系統 (Discharge Referral Management System)

## 功能說明

### 主要功能
1. **患者資料管理**
   - 顯示患者基本資料（病歷號、床號、姓名、性別、年齡等）
   - 搜尋和篩選功能
   - 新增、編輯、刪除患者記錄
   - 實時統計患者數據

2. **評分系統**
   - 點擊任意患者資料行即可開啟「出院準備服務篩選表」
   - 篩選表包含多個評估項目（婚姻狀況、出院照護、醫療照護、健康問題等）
   - 系統自動計算總分
   - 評分結果會顯示在主頁面的「評分」欄位

3. **資料匯出**
   - 可將資料匯出為 CSV 格式

4. **現代化設計**
   - 深色主題科技感設計
   - 霓虹青色主題配色
   - 玻璃態效果
   - 流暢的動畫過渡
   - 完全響應式設計

## 使用方式

### 評估患者
1. 在主頁面點擊任一患者資料行（或點擊評分按鈕）
2. 系統會彈出「出院準備服務篩選表」視窗
3. 在表單中選擇相關評估項目（使用 radio button）
4. 系統自動計算總分並顯示
5. 點擊「儲存評分」按鈕保存
6. 返回主頁面後，評分欄位會更新顯示總分

### 操作說明
- **新增**：新增患者記錄
- **刪除**：勾選後批次刪除
- **搜尋**：輸入關鍵字即時搜尋
- **篩選**：按轉銜狀況篩選
- **保存**：保存編輯的資料（Ctrl+S）
- **匯出**：匯出 CSV 檔案

### 開啟系統
1. 直接在瀏覽器中打開 `index.html`

## 注意事項

1. 評分資料保存在瀏覽器的 LocalStorage 中
2. 點擊患者資料行的任意位置（除複選框和按鈕）都會開啟評估表單
3. 評分標籤顏色說明：
   - 紫色漸層：已評分
   - 灰色：未評分
4. 評估表單中的分數會自動累加計算
